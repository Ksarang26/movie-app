# 🎬 MERN Mastery: Building a Scalable Movies App for Ultimate User Experience

A full-stack movie application built using the MERN stack, focusing on scalability, authentication, and a modern user experience.

---

## 🚀 Tech Stack

### Frontend

* React.js
* Vite
* Tailwind CSS
* React Router
* Axios

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT Authentication
* bcrypt.js
* Cookie Parser

---

## ✨ Features

* User Registration
* User Login & Logout
* JWT Authentication
* Protected Routes
* Admin Authorization
* Movie Management
* Genre Management
* Reviews & Ratings
* Responsive UI
